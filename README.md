This Python-based command-line tool is a real-time currency exchange monitoring application designed to fetch, parse, and compare financial data from live banking web services. Developed as part of a computer science curriculum, the script automates HTTP GET requests to interface directly with the public REST APIs of PrivatBank and Monobank, two of Ukraine's major financial institutions. The backend core engine safely processes JSON responses, mapping international ISO currency codes (such as 840 for USD, 978 for EUR, and 980 for UAH) and handling varying API data structures—including standard buy/sell rates and cross-currency exchange parameters. Once the payloads are successfully retrieved and normalized, the application utilizes programmatic string formatting to build a clean, structured matrix directly within the terminal. This layout manager displays side-by-side evaluations of purchasing and selling prices for both US Dollars and Euros up to four decimal places, allowing users to immediately spot market variance. By integrating external Python packages like requests and focusing on dynamic data parsing, error-resistant collection mapping, and clear user feedback, the project provides a foundational example of network communication and API integration in commercial-grade script development.
